# Spaces by Rogue Divisions — landing page

Live: https://spaces.roguedivisions.com
Host: Cloudflare Workers (static assets from `./public`)
Worker name: `rd-spaces`

## Publish flow

1. Edit the source files, not generated segment HTML:
   - `src/template.html` for shared markup and runtime JavaScript
   - `src/segments.json` for per-segment content and calculator config
   - `public/spaces.css` for styles
   - `public/_headers` for Cloudflare static-asset headers
2. Build generated pages with `python3 scripts/build.py`.
3. Verify with `python3 scripts/verify.py`. If Playwright Chromium is missing, run `python3 scripts/setup_playwright.py` once.
4. `git add . && git commit -m "<change>" && git push`
5. If GitHub auto-deploy is wired up, that is it. Otherwise:

   ```
   set -a && source /path/to/your/cloudflare.env && set +a   # CLOUDFLARE_API_TOKEN=...
   npx wrangler@latest deploy
   ```

## Notes

- `public/index.html` and `public/<segment>/index.html` are generated by `scripts/build.py`. Do not edit them directly.
- `noindex, nofollow` meta tag is set — page is unlisted, by-referral only.
- DNS is managed automatically via `custom_domain = true` in `wrangler.toml`.
- Deploy requires a Cloudflare API token in your environment (`CLOUDFLARE_API_TOKEN`).
